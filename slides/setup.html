Setup

  • Download IntelliJ Idea (CE) from https://www.jetbrains.com/idea/download/
  • Install IntelliJ with Scala support
  • Download & Install a Git client
  • Download & Install SBT from https://www.scala-sbt.org/download.html
  • Clone repository
  • Create some class or worksheet and run a simple Hello World application to check everything is OK
